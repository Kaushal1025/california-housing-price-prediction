# California Housing Price Prediction

Built a machine learning model using XGBoost and Streamlit to predict California housing prices based on housing features from the California Housing dataset.

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost
- Streamlit
- Joblib

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Correlation heatmaps and feature selection
- XGBoost regression model training
- Model evaluation and optimization
- Interactive Streamlit web app for user predictions

## Outcome
Improved prediction accuracy and created a user-friendly interface where users can input housing details and receive real-time price predictions.
